title: Playground
description: Build and test UI payloads live against a real device.
---

Use the playground to prototype member states without writing any host code. Create a room, scan the code with your phone (or open it in another tab), and edit the JSON below — every change is pushed to the connected device instantly. It's the fastest way to dial in [components](/components/overview), [theming](/advanced/theming), and [presets](/advanced/presets) before wiring them into your game.

::playground
