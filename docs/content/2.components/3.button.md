A button that sends a fixed `value` back to your host when tapped.

```json
{
  "type": "Button",
  "props": {
    "event": "answer",        // REQUIRED — the event name sent back in msg
    "label": "A: 42",          // button text (supports Markdown)
    "value": "A",              // the value sent back in msg
    "keys": ["A", "1"],        // optional keyboard keys that also trigger it
    "persistent": false,       // if true, stays enabled for repeated presses
    "style": {
      "color": "black",
      "background": "#AAAAAA",
      "border": "2px solid black",
      "width": "100%",
      "fontSize": "20px",
      "padding": "0 20px",
      "margin": "10px 0px",
      "borderRadius": "10px",
      "fontFamily": "sans-serif",
      "hover": { "color": "black", "background": "#888888" }
    }
  }
}
```

| Prop | Type | Notes |
| --- | --- | --- |
| `event` | string | **Required.** The event name your host receives in the `msg`. |
| `label` | string | Button text. Supports Markdown. |
| `value` | string | The value sent back when tapped. |
| `keys` | string[] | Optional keyboard keys that also trigger the button. |
| `persistent` | boolean | Default `false`. When `true`, stays enabled for repeated presses. |
| `style` | object | Standard [style object](/components/overview#the-style-object), plus a nested `hover` object. |

**Sends:** `msg` on tap, with `value` set to the `value` prop (and `ms`, the time since the button appeared). By default the button disables after one press to prevent double-submits; set `persistent: true` to allow repeated presses.
