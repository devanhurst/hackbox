---
title: Text
description: Display static text, with Markdown support.
---

Display static text. Supports **Markdown** (and inline HTML like `<br />`). Display only — `Text` sends nothing back to your host.

```json
{
  "type": "Text",
  "props": {
    "text": "**Round 1**<br />Get ready!",
    "style": {
      "color": "black",
      "textAlign": "center",
      "background": "white",
      "border": "4px solid black",
      "fontSize": "16px",
      "padding": "10px",
      "margin": "10px 0px",
      "borderRadius": "10px",
      "fontFamily": "sans-serif"
    }
  }
}
```

| Prop | Type | Notes |
| --- | --- | --- |
| `text` | string | The content. Markdown and inline HTML are rendered. |
| `style` | object | Standard [style object](/components/overview#the-style-object). |
