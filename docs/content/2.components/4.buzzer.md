title: Buzzer
description: A big buzz-in button built for "first to buzz" moments.
---

A big circular buzz-in button, triggered by tap or the **Space**/**Enter** key. Built for "first to buzz" moments — the `ms` reaction time is the point.

```json
{
  "type": "Buzzer",
  "props": {
    "label": "BUZZ",        // default: "BUZZ"
    "event": "buzz",         // default: "buzz"
    "style": {
      "color": "white",
      "background": "red",
      "boxShadow": "5px 5px #000000",
      "borderRadius": "70px",
      "fontSize": "70px",
      "height": "300px",
      "border": "2px solid white",
      "fontFamily": "Fredoka One"
    }
  }
}
```

| Prop | Type | Notes |
| --- | --- | --- |
| `label` | string | Default `"BUZZ"`. |
| `event` | string | Default `"buzz"`. |
| `style` | object | Standard [style object](/components/overview#the-style-object). |

**Sends:** `msg` once on press, with `value: null` — read `ms` for the reaction time. The buzzer disables after a press; send the player a new state to re-arm it.
