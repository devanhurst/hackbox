A numeric answer via a draggable slider, paired with a number input.

```json
{
  "type": "Range",
  "props": {
    "event": "estimate",
    "min": 0,        // default: 0
    "max": 100,      // default: 100
    "step": 1,       // default: 1
    "style": {
      "color": "black",
      "background": "white",
      "border": "2px solid black",
      "width": "100%",
      "fontSize": "16px",
      "padding": "10px",
      "margin": "10px 0",
      "fontFamily": "sans-serif"
    }
  }
}
```

| Prop | Type | Notes |
| --- | --- | --- |
| `event` | string | The event name your host receives. |
| `min` / `max` | number | Range bounds. Default `0` / `100`. |
| `step` | number | Increment. Default `1`. |
| `style` | object | Standard [style object](/components/overview#the-style-object). |

**Sends:** `change` as the player drags, and `msg` on submit — both with `value` set to the chosen number (as a string).
