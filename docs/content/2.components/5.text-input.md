title: TextInput
description: A text field that sends its contents back on submit.
---

A text field that sends its contents back to your host on submit. Set `type: "number"` to restrict it to numbers.

```json
{
  "type": "TextInput",
  "props": {
    "event": "guess",
    "type": "text",          // "text" or "number"
    "min": 0,                 // number type: minimum
    "max": 100,               // number type: maximum
    "step": 1,                // number type: step (default: 1)
    "persistent": false,      // if true, clears and allows another submission
    "style": {
      "color": "black",
      "background": "white",
      "border": "2px solid black",
      "width": "100%",
      "fontSize": "30px",
      "padding": "10px",
      "margin": "10px 0",
      "fontFamily": "sans-serif"
    }
  }
}
```

| Prop | Type | Notes |
| --- | --- | --- |
| `event` | string | The event name your host receives. |
| `type` | string | `"text"` (default) or `"number"`. |
| `min` / `max` | number | Bounds for `number` type. |
| `step` | number | Step for `number` type. Default `1`. |
| `persistent` | boolean | Default `false`. When `true`, the field clears after each submit so the player can send several values. |
| `style` | object | Standard [style object](/components/overview#the-style-object). |

**Sends:** `change` as the player types, and `msg` on submit — both with `value` set to the entered string.
